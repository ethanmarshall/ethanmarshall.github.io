HTML Structure and Tags (Syntax)

HTML documents have a fixed structure and an array of standard tags and other elements which are fluidly used in the makeup of each document. While some tags are no longer used (deprecated) you may still find some in use today. Here is a list of various HTML tags, their names, and what they do/how we use them in in the construction of a well made HTML document.

HTML Element Name Description & Use
<!DOCTYPE> Document Type Defines that this document is an HTML5 document
<html> HTML Root element of an HTML page
<head> Head Contains meta (self referral) info about the HTML page such as stylesheet, font usage, tab title and others
<title> Title Specifies a title for the HTML page (shown in the title bar/tab)
<body> Body Defines the document body and contains all visible components of the HTML page
<h1/h2/h3/h4/h5/h6> Heading Defines a heading over various sizes from h1-h6 from largest to smallest
<p> Paragraph Defines text as a paragraph
<link> Link Used to link to an external resource, such as a CSS file or webfonts
<main> Main Contains the most important content of your page
<footer> Footer At the bottom of your page, contains various contact information, copyright info, and about info
<header> Header Page headline. Typically contains page title and useful links for page navigation
<nav> Navigation Contains the navigation links for your page/website
<a> Anchor Used to create links to another part of your page
<article> Article Self-contained text such as news articles or large paragraphs
<aside> Aside Used in conjunction with the article tag for things such as quotes or other useful info
<ul> Unordered List Used to create a bulleted list
<ol> Ordered list Used to create a numbered list
<li> List Item Contains the actual item/info to be contained in the ordered or unordered list
<img> Image Contains reference to an image. <img src="..."/> Self closing tag.
<video> Video Contains reference to a video. Unlike <img>, this tag is not self closing.
<embed> Embed Used to embed a gif, video, or other media on a page. Deprecated tag.
<audio> Audio Used to add audio to a page.
<div> Division Used to divide portions of your documents main body
<span> Span Used to allow seprate formatting of content whilst maintaining the text inline
<alt> Alternate Text Specifies an alternate description for an image if it cannot be displayed or for accessibility

For a more extensive reference on HTML tags and structure, visit MDN

HTML Tables

You can build HTML tables using these tags and some intuitive structuring of each.

Tag Name Description
<table> Table The container for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing the actual table data.
<tr> Table Row The table rows.
<td> Table Data The table data that fills the rows.
<tfoot> Table Foot The set of rows defining the table footer.
Attribute Name Description
<colspan> Column Span Defines how many columns a <td> element should span.
<rowspan> Row Span Defines how many rows a <td> element should span.

For a more extensive refence on HTML tables and their structure, visit MDN

CSS Styling

Styling of your page can be done inline in your HTML code (not recommended), or via CSS stylesheets. Cascading Stylesheets (CSS) are seperate files that you can connect to your index.html file and apply various stylings such as text editing to include font colors, weights, familys, background images, and much more. To connect it to your html file, you must add a reference in the "head" of your document with the following syntax:
<link rel="stylesheet" href='css/stylesheet.css' type="text/css">. Your reference doesn't have to match that exactly, but must make sense.

This styling is applied via different selectors, with varying degrees of specificity. Here is the order of precedence:

  1. Inline styles. This is referring to applying your styling via the HTML document directly.
  2. IDs. Applied to an element in the same manner as classes, but more specific.
  3. Classes, pseudo-classes, attribute selectors.
  4. Elements and pseudo-elements.

Nested elements: styling is applied to child elements, e.g. the list items of an <ol> or <ul></>

Inheritance is applying styling from the topmost html element to all other child elements. E.g. styling applied to <html> applies to all intermediate elements, unless more specific styling is present in the CSS.

There are several ways to chain selectors for styling and the chaining method depends on the type of selector.
